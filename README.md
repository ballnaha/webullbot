# 🚀 Webull Trading Bot - Python

บอทเทรดหุ้นอัตโนมัติและจำลองการเทรด (Simulation) สำหรับ Webull พัฒนาด้วยภาษา Python พร้อมหน้าจอคอนโซลควบคุม (Interactive CLI Dashboard) ที่สวยงามด้วยไลบรารี `rich`

---

## 🌟 ฟีเจอร์หลัก (Features)
- **4 โหมดการเทรด (Trading Modes)**:
  1. **LOCAL_PAPER**: เทรดจำลองแบบออฟไลน์ ดึงข้อมูลราคาจาก Yahoo Finance (`yfinance`) อัปเดตพอร์ตจำลองโดยไม่ต้องใช้ API Key หรือรหัสผ่านใดๆ ปลอดภัยสำหรับทดลองบอท
  2. **WEBULL_PAPER**: เทรดจำลองบนระบบจำลอง (Paper Trading) จริงของ Webull (จำเป็นต้องใช้บัญชีในการล็อกอิน)
  3. **WEBULL_LIVE**: เทรดเงินจริงผ่าน Webull (ต้องเข้าสู่ระบบและระบุ Trade PIN) *คำเตือน: โปรดใช้ด้วยความระมัดระวัง*
  4. **WEBULL_OFFICIAL**: เชื่อมต่อผ่าน OpenAPI ทางการของ Webull (สำหรับผู้มี App Key และ App Secret)
- **กลยุทธ์ซื้อขายอัตโนมัติ (Strategy Engine)**:
  - **SMA Crossover (เส้นค่าเฉลี่ยตัดกัน)**: ซื้อเมื่อ SMA ระยะสั้น (เช่น 10) ตัดขึ้นเหนือ SMA ระยะยาว (เช่น 30) และขายเมื่อตัดลง
  - **RSI Strategy**: ตรวจจับดัชนีกำลังสัมพัทธ์ ซื้อเมื่อ RSI กลับตัวพ้นเขตขายมากเกินไป (Oversold < 30) และขายเมื่อ RSI หลุดจากเขตซื้อมากเกินไป (Overbought > 70)
- **Terminal UI Dashboard**: หน้าจอแดชบอร์ดแสดง ยอดเงินคงเหลือ, รายการหุ้นที่ครอบครอง (Positions), ล็อกการทำงานของบอท และประวัติการทำธุรกรรมแบบเรียลไทม์

## 🛠️ ขั้นตอนการติดตั้ง (Installation)

1. ตรวจสอบให้แน่ใจว่าติดตั้ง Python 3.8 - 3.11 เรียบร้อยแล้ว
2. เปิด Terminal ในโฟลเดอร์ของบอท และติดตั้งโมดูลที่จำเป็น:
   ```bash
   # หากใช้ Windows ภาษาไทย หรือพบปัญหา encoding ในการติดตั้งให้ใช้คำสั่งนี้:
   $env:PYTHONUTF8=1; pip install -r requirements.txt
   
   # หรือคำสั่งปกติ:
   pip install -r requirements.txt
   ```

---

## 🚀 วิธีการใช้งาน (Usage)

1. รันไฟล์หลักด้วยคำสั่ง:
   ```bash
   python main.py
   ```
2. เลือกเมนู **[3] Setup Wizard / Configuration** เพื่อตั้งค่าบัญชีและพารามิเตอร์การเทรดผ่านหน้าจอแนะนำตัวช่วย ระบบจะสร้างไฟล์ `.env` ให้คุณโดยอัตโนมัติ
3. สำหรับการทดสอบโดยไม่มีความเสี่ยง ให้เลือกโหมด `LOCAL_PAPER` ระบบจะสร้างไฟล์เก็บสถานะพอร์ตชื่อ `local_portfolio.json` ไว้ในเครื่องของคุณ
4. เลือกเมนู **[1] Start Trading Bot** และเลือกกลยุทธ์ที่ต้องการใช้งาน บอทจะเริ่มทำงาน วาดหน้าจอ Dashboard และวิเคราะห์สัญญาณซื้อขายตามช่วงเวลา (Interval) ที่ตั้งค่าไว้

---

## 📁 โครงสร้างโฟลเดอร์ของโปรเจกต์
- `main.py`: จุดเข้าใช้งานหลักและหน้าจออินเตอร์เฟสหลักของบอท
- `bot.py`: ตัวประมวลผลการทำงานหลักของบอท ดึงราคา คำนวณสัญญาณ และส่งคำสั่งเทรด
- `config.py`: จัดการและตรวจสอบตัวแปรคอนฟิกจากไฟล์ `.env`
- `clients/`: ตัวเชื่อมต่อ API ในรูปแบบต่างๆ (จำลองในเครื่อง, Webull Unofficial, Webull Official)
- `strategies/`: ไฟล์คำนวณสูตรเทคนิคอลและส่งสัญญาณซื้อขาย (SMA Crossover, RSI)
- `requirements.txt`: รายการไลบรารีที่บอทต้องการ
